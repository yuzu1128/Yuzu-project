GitHub Pages 有効化手順
=====================================

ゆづさん、もうすぐ完成です！
あと少しだけ手動設定が必要です。

【GitHub Pagesを有効化する手順】

1. 以下のURLにアクセス：
   https://github.com/yuzu1128/Yuzu-project

2. 「Settings」タブをクリック
   （リポジトリ名の下にあるメニューバー）

3. 左側メニューから「Pages」をクリック

4. 「Source」セクションで：
   - Source: 「Deploy from a branch」を選択
   - Branch: 「main」を選択
   - Folder: 「/ (root)」を選択

5. 「Save」ボタンをクリック

6. 1-2分待つ

7. 以下のURLでアクセス可能になります：
   
   🌟 あなたの永続的URL：
   https://yuzu1128.github.io/Yuzu-project/

=====================================

=====================================

   「Your site is live at...」が表示されたら成功

2. URLをクリックして動作確認
